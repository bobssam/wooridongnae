공통 리스트

[apiName]/list

리스트

리스트형 데이터를 반환합니다.

요청

{
	"page":1,
	"len":10,  // 옵션입니다.
	"filter": {}, // 옵션입니다.
	"oreder": [{"idx":"desc"}], // 옵션입니다.
}

page 페이지는 뭡니다
len 로딩할 길이 입니다.
filter 검색어를 입력하여서 필요한 데이터만 요청합니다.
order 정렬 조건을 추가합니다.

정상응답

{
	"status" : "ok",
	"total": 100,	// 옵션
	"list" : [{"key":"value"}]
}

total 전체 글 크기 입니다. API에 따라선 총길이를 반환하지 않아도 됩니다.
list 리스트 내용입니다. 내용물은 API 마다 다릅니다.

에러응답 - 데이터 없음

{
	"status": "fail",
	"reason":"데이터가 없습니다."
}
