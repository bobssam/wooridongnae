공통 수정

[apiName]/modify

수정

글이나 데이터를 수정합니다.

요청

{
	"where": {
		// 검색 조건
	},
	"data": {
		// 갱신할 데이터
	}
}

where 복합적인 조건을 보낼수도 있습니다만, API마다 조건은 선택적으로 이용합니다.
data 갱신 데이터를 보냅니다. 모든 데이터를 갱신할지는 API마다 다릅니다.

정상응답

{
	"status" : "ok",
	"affectedRow" : 1	// 지원진 row 수
}

에러응답 - 실패

{
	"status": "fail",
	"reason":"[상황에 맞는 안내]"
}
