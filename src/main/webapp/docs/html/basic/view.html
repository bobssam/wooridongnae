[apiName]/view

상세보기

지정한 한가지 글에 대한 상세 정보를 줍니다.

요청

{
	"where" : {
		"[pk_column]":12
	}
}

where 선택 조건을 넣습니다. 보통은 PK값을 입력합니다.

정상응답

{
	"status" : "ok",
	"data" : {"key":"value"}
}

status 응답의 상태입니다.
data 본문의 내용입니다. 내용물은 API 마다 다릅니다.

에러응답 - 데이터 없음

{
	"status": "fail",
	"reason":"데이터가 없습니다."
}
